
TOPIC
    about_Send-Message

SHORT DESCRIPTION

    Send message to discord webhook.

PARAMETER Content
    
    <string>
    Message to send.

PARAMETER Username

    <string>
    Discord username.

PARAMETER Webhookuri

    <string>
    Enter webhook https address.
    Example: 'https://discord.com/api/webhooks/restofyourwebhookhere'.

EXAMPLES

    Send-Message -Content "Script Finished." -Username "discordusername" -Webhookuri "https://discord.com/api/webhooks/restofyourwebhookhere"

KEYWORDS

    Send-Message Message Send Discord Webhook
